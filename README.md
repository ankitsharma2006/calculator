# Calculator Web App

## Overview
This is a simple calculator web application built using HTML, CSS, and JavaScript. It performs basic arithmetic operations with a clean and user-friendly interface.

## Features
- Basic operations: addition, subtraction, multiplication, division
- Clear button to reset the display
- Error handling for invalid inputs
- Result persists until manually cleared (does not reset on refresh)

## How It Works
- Users click buttons to input numbers and operators
- The input is displayed in real-time
- Press "=" to evaluate the expression
- Press "Clear" to reset the calculator

## Technologies Used
- HTML
- CSS
- JavaScript

## Usage
1. Clone or download the repository
2. Open `index.html` in your browser
3. Start using the calculator

## Notes
- Uses JavaScript for dynamic evaluation of expressions
- Basic error handling is implemented for invalid expressions
